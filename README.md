# FairPlayAI

AI-powered bias & fairness checker for ML models.  
Built with Django, React, Node.js, and scikit-learn.

## Features
- Bias metrics (Disparate Impact, Demographic Parity)
- Upload models/datasets
- Visual dashboards
- MongoDB + JWT auth

## Tech Stack
- Frontend: React (Vite)
- Backend: Django + Node.js
- DB: MongoDB
